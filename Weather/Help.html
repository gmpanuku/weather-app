Cities


A list of saved cities added/shown on this page where a search option is available to find popular cities (i.e Hyderabad, Chennai, New Delhi, etc..).

  • Add Location: To add a location click on the + button on the left side of the toolbar which navigates to the map page. And user can search the city by its name in the search bar.


  • Remove Location: The user can delete the location by swiping to the right side of any location.             


  • Check Weather Details: The user can click on any location which was added, It will navigate to the Weather details page, Where today's weather details are displayed.

Map


Users can long press anywhere on the map, It will display a popup with the Location Name and Cancel and Add options. 

Click on Add button to add the location and then the location will be added and it takes back to the cities page, Where the location was added.


City Weather Details


On this page, today's weather details like Temperature, Weather condition, Max temperature, Min temperature, Humidity, Sunrise, Sunset, Wind speed, Air pressure, Visibility are displayed.


Settings


On this page, the user can change the temperature units(i.e metric/imperial) and also reset the added cities.
